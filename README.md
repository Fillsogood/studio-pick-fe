# Studio Pick Frontend

React + Vite + Tailwind CSS로 구성된 프론트엔드 프로젝트입니다.

## 기술 스택

- **React 18** - 사용자 인터페이스 라이브러리
- **Vite** - 빠른 빌드 도구
- **Tailwind CSS** - 유틸리티 우선 CSS 프레임워크
- **JavaScript** - 프로그래밍 언어
- **ESLint** - 코드 품질 도구

## 시작하기

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

서버가 시작되면 `http://localhost:3000`에서 애플리케이션을 확인할 수 있습니다.

### 빌드

```bash
npm run build
```

### 프리뷰

```bash
npm run preview
```

### 린팅

```bash
npm run lint
```
